## üÜò EMERGENCY

```powershell
# Check what's wrong
docker compose logs appwrite

# Full health scan
.\monitor-cloud-health.ps1 -Command health

# Restart everything
docker compose restart

# Clean rebuild (loses cache)
docker compose down
docker compose up -d

# Restore from backup
.\monitor-cloud-health.ps1 -Command restore -RestoreFile "path/to/backup.zip"
```
